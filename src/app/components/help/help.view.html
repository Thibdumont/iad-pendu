Manuel du jeu

  1. Renseignez votre nom sur la page d'accueil
  2. Choisissez la difficulté dans laquelle vous souhaitez jouer
  3. Trouvez le mot en choisissant des lettres de l'alphabet
  4. Si la lettre choisie est dans le mot, ces lettres apparaissent
  5. Si la lettre n'est pas dans le mot, une pièce du pendu est ajoutée
  6. Vous perdez la partie une fois le pendu totalement construit (après 10 tentatives)
